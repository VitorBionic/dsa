# Data Structures and Algorithms C Library

A lightweight and reusable Data Structures and Algorithms library written in C. It provides implementations of fundamental data structures, including linked lists, stacks, queues, sets, hash tables, trees, and others, designed to be used in larger and more complex C projects.

## Building

First, clone the repository:

```sh
git clone https://github.com/VitorBionic/dsa.git
cd dsa
```

Then build the project:

```sh
make
```

The shared (`.so`) and static (`.a`) libraries will be generated in:

```text
build/release/lib/
```
